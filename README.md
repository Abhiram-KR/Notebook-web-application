# Notebook – Personal Notes Management System

Notebook is a web-based personal notes management system designed to help users securely create, organize, and manage their notes online. The application focuses on simplicity, security, and usability, making it suitable for students and general users.

This project is currently under development as an academic mini-project.

---

## Project Status
**Work in Progress**

---

## Features

- Secure user registration and login
- OTP-based password recovery
- Create, edit, view, and delete notes
- Pin important notes for quick access
- Archive and restore notes
- Password-protected notes and notebooks
- Trash management for deleted notes
- Admin view for users and feedback (basic level)

---

## Technologies Used

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** Supabase (PostgreSQL)
- **Architecture:** MVC-style (Model–View–Controller)
- **Tools:** Visual Studio Code / Any IDE

---

## System Requirements

### Hardware
- Processor: Intel i3 or higher
- RAM: Minimum 4 GB
- Storage: Minimum 10 GB free space
- Internet connection

### Software
- OS: Windows / Linux / macOS
- Browser: Chrome, Firefox, or Edge

---

## Objective

The objective of this project is to build a secure, user-friendly platform that balances productivity and privacy, enabling users to manage personal notes efficiently while adhering to proper software engineering practices.
